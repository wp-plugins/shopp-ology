You can then share this information with your web developer, Shopp consultant, or responding Shopp agent on the Help Desk.

Ensure that your [Shopp](http://shopplugin.net) is activated and up-to-date.2. Download Shopp + Ology. Unzip the `shopp-ology` folder.3. Upload the `shopp-ology` folder to the `wp-content/plugins/` directory.4. Activate Shopp + Ology through the Plugins menu in the WordPress admin.5. Go to Shopp &rarr; + Ology.
