You can then share this information with your web developer, Shopp consultant, or responding Shopp agent on the Help Desk.

Ensure that your [Shopp](http://shopplugin.net) is up-to-date.2. Upload the `shopp-ology` directory to the `wp-content/plugins/` directory.3. Activate the plugin through the 'Plugins' menu in WordPress.4. Go to Shopp &rarr; + Ology.
